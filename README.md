
# Woman ✨
Woman is a lightweight AI-powered CLI tool that explains terminal commands instantly using Google Gemini ⚡

## 🚀 What it does
Just pass any command and get a quick, simple explanation

It returns:

📌 What the command does  
⚡ How to use it  
🔧 Common flags  
💡 A simple example  

## 🧑🏻‍💻 Example

    woman git # Explains git command

## 📦 Installation

    go install github.com/FardinEbrahimix/woman@latest

**This tool uses Google Gemini API 🤖**  
Get your API key: https://aistudio.google.com/
